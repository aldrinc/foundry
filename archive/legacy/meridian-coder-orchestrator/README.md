# Meridian Coder Orchestrator (Legacy)

Archived: 2026-03-13

This package preserves sanitized legacy Meridian orchestrator source for
reference during the Foundry migration. It is not the active Foundry runtime.

Excluded on purpose:

- `.env` and local config files
- databases and runtime state
- caches, venvs, and generated artifacts
- memory files and other private local data
